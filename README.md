# 🚀 Registration System (CRUD) with Python & SQLite

This is a practical project of a user management system via terminal. The main focus was to apply concepts of **Data Persistence** and **Programming Logic** in Python.

## 📋 Features
- **Create:** Adds a user's name and age directly into the database.
- **Read:** Queries and displays all saved records in an organized way.
- **Delete:** Removes a specific user using their unique ID.
- **Error Handling:** Basic validation to prevent invalid data input (e.g., letters in the age field).

## 🛠️ Technologies Used
- **Python 3:** Main programming language.
- **SQLite3:** Lightweight relational database (built into Python), eliminating the need to install external servers for this project.

## 📁 Project Structure
- `main.py`: Contains all the system logic and database connection.
- `database.db`: File automatically generated to store data persistently.

## 🚀 How to Run
1. Make sure you have Python installed.
2. Clone this repository or download the main.py file.
3. In the terminal, run:
   ```bash
   python main.py
