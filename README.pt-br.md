# 🚀 Sistema de Cadastro (CRUD) com Python & SQLite  

Este é um projeto prático de um sistema de gerenciamento de usuários via terminal. O foco principal foi aplicar conceitos de **Persistência de Dados** e **Lógica de Programação** em Python.  

## 📋 Funcionalidades  
- **Criar:** Adiciona o nome e a idade de um usuário diretamente no banco de dados.  
- **Ler:** Consulta e exibe todos os registros salvos de forma organizada.  
- **Deletar:** Remove um usuário específico utilizando seu ID único.  
- **Tratamento de Erros:** Validação básica para evitar entrada de dados inválidos (ex.: letras no campo de idade).  

## 🛠️ Tecnologias Utilizadas  
- **Python 3:** Linguagem de programação principal.  
- **SQLite3:** Banco de dados relacional leve (embutido no Python), eliminando a necessidade de instalar servidores externos para este projeto.  

## 📁 Estrutura do Projeto  
- `main.py`: Contém toda a lógica do sistema e a conexão com o banco de dados.  
- `database.db`: Arquivo gerado automaticamente para armazenar os dados de forma persistente.  

## 🚀 Como Executar  
1. Certifique-se de ter o Python instalado.  
2. Clone este repositório ou baixe o arquivo main.py.  
3. No terminal, execute:  
   ```bash
   python main.py
